# NimitAI Intern Assignment

## Overview

A web application that analyzes sales meeting transcripts and detects:

- Buying Interest
- Objections
- Confusion

It also generates a coaching tip for each detected signal.

## Tech Stack

### Frontend
- React
- Vite
- Axios

### Backend
- Node.js
- Express.js

### LLM Used
- Gemini 2.5 Flash (Google Generative AI)
